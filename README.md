# Smart Motion-Triggered Alarm System
University of Florida – EEL3303 Final Design Project  
Author: Luca Cabiddu

This Arduino project uses a tilt switch, RGB LED, buzzer, LCD, and push button to create a motion-triggered alarm. The system switches between DISARMED, ARMED, and ALARM states and displays status updates on a 16×2 LCD.
